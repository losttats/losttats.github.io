自定义弹窗

主要思路是弄一个半透明的遮罩

#bc{
    display: none;
    position: fixed;
    left: 0;
    top:0;
    width: 100%;
    height:100%;
    background-color: rgba(0,0,0,0.5)
}

定宽高比

padding的百分比是相对于其包含块的宽度来确定的

div{
    width: 100px;
    background: red;
    display: flex;
}   
div:after{
        content: '';
        padding-top: 100%;
}

  • 字体属性系列:font-family,font-weight,font-size,font-style

  • 文本属性系列:text-indent,text-align,line-height,word-spacing,letter-spacing,color

  • 元素可见性:visibility

  • 列表属性系列:list-style

  • 光标:cursor

三栏布局
